# Relatório da Atividade

## 1. Objetivo

Criar um projeto React contendo um cabeçalho com as funcionalidades pedidas na atividade e implementar cada recurso em uma interface única e responsiva.

## 2. Funcionalidades implementadas

### To-Do List
Permite adicionar e remover tarefas de forma dinâmica.

### Contador de Cliques
Exibe um contador simples com botões para incrementar e zerar.

### Jogo da Velha
Implementa um jogo funcional com detecção de vencedor e reinício da partida.

### Calculadora
Permite montar expressões básicas e calcular resultados diretamente na interface.

### Buscador de CEP
Consulta a API pública do ViaCEP e mostra os dados do endereço retornado.

## 3. Estilização escolhida

A interface foi estilizada com uma proposta clara e moderna, usando:

- cartões com efeito translúcido;
- fundo em degradê com formas suaves;
- botões arredondados e sombras leves;
- layout responsivo em grid;
- tipografia com foco em legibilidade.

A intenção foi deixar a página organizada e visualmente mais agradável, sem comprometer a simplicidade necessária para a atividade.
