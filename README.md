# Tarefa 06 - Projeto React

Projeto React desenvolvido para a atividade com cinco funcionalidades: To-Do List, Contador de Cliques, Jogo da Velha, Calculadora e Buscador de CEP.

## Como executar

1. Instale as dependências:

```bash
npm install
```

2. Inicie o projeto:

```bash
npm run dev
```

3. Abra o endereço exibido no terminal.

## Estrutura criada

- `src/App.jsx`: concentra as cinco funcionalidades da interface.
- `src/main.jsx`: ponto de entrada do React.
- `src/styles.css`: estilização responsiva e visual principal do projeto.
- `docs/relatorio.md`: documento de entrega da atividade.

## Observação

O projeto usa a API pública do ViaCEP para consultar endereços no buscador de CEP.
